# CardPilot

Personal-use credit card tracker built with Expo React Native.

## Run

1. Install dependencies:

```bash
npm install
```

2. Start Expo:

```bash
npm start
```

3. Open in:
- iOS Simulator with `i`
- Android Emulator with `a`
- Expo Go by scanning the QR code


## Current MVP

- Cards tab with all cards in one place
- Best Card recommendations by category
- Benefits tab for unused credits
- Local-only storage with AsyncStorage
- Card detail view
- Add card from built-in templates
- Mark benefits used or reset them

## Notes

- Notifications are not wired yet in this React Native version.
- Data is seeded locally on first launch using built-in sample cards.
